# Home Page Images

Place home page images in this directory:

- `hero-education.svg` - **SVG placeholder created** - Main hero section image
  - Currently using an SVG placeholder with educational icons
  - You can replace with: `hero-education.jpg` or `hero-education.png`
  - Recommended size: 1200x800px or larger
  - Format: JPG, PNG, or WebP
  - Aspect ratio: 3:2 (landscape)
  - Should show students, education, or counseling scene

- `education.jpg` - Main education/counseling image for the "Why Choose Us" section
  - Recommended size: 1200x800px or larger
  - Format: JPG, PNG, or WebP
  - Aspect ratio: 3:2 (landscape)

**Note:** The SVG placeholder (`hero-education.svg`) is currently being used. If you add a JPG/PNG file with the same name, it will automatically be used instead. The SVG includes educational icons and gradients matching your brand colors.
